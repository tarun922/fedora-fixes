PERFORMANCE

Speed up DNF

Fedora's package manager is safe but slow by default. Enable parallel downloads and fastest mirrors.

sudo nano /etc/dnf/dnf.conf
# Add: max_parallel_downloads=10
MULTIMEDIA

Media Codecs

Install essential codecs (H.264, AAC, etc.) that aren't included by default due to licensing.

sudo dnf groupupdate core
SYSTEM

Battery Life (TLP)

Significantly improve laptop battery life by installing TLP power management.

sudo dnf install tlp tlp-rdw

About this Project

Fedora Fixes was born out of the need for a central repository of post-installation steps. Instead of scouring Reddit and various forums, we've compiled the most critical updates to get your system running at 100% efficiency in minutes.
